# ①課題番号-プロダクト名

ランチメモアプリ

## ②課題内容（どんな作品か）

- ランチを食べたレストランの記録
- 評価、感想、店情報(url)を記録できる。
- 記録を見るときに、カテゴリー(和食、中華など)別に表示できる
- 記録をみるときに、日付順、評価順に表示できる
- 評価、感想の更新ができる。
- 削除ができる
